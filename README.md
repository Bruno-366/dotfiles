# dotfiles
My dotfiles for [ArchLinux]

## Desktop Enviroment

0. bash with [`.bashrc`] or [`.bash_profile`]
1. startx (wrapper around xinit) with file [`.xinitrc`]
2. i3wm with file 
   1. surf
   2. ranger

[`.bashrc`]: ./.bashrc
[`.bash_profile`]: ./.bash_profile
